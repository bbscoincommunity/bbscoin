# f\_block\_json

### Input

| Argument | Mandatory | Description                      | Format |
| -------- | --------- | -------------------------------- | ------ |
| hash     | yes       | Block hash of the block to query | string |

### Output

| Argument | Description                               | Format |
| -------- | ----------------------------------------- | ------ |
| block    | Block details                             | binary |
| status   | Status of the request, "OK" if successful | string |
